# Release Notes: 1.9.8-alpha

**Artifact:** `deployment/prospecting-extension-1.9.8-alpha.zip`
**Integrity sidecar:** `deployment/prospecting-extension-1.9.8-alpha.zip.sha256`

## Phase3C17 WP1.4 product navigation polish

- Consolidates the product IA into Chinese-first navigation sections for
  prospecting, customer management, activity, and general tools.
- Keeps `Lead` as one native global entry and retains supporting records behind
  their existing operational-center, relationship-panel, or dashboard paths.
- Adds complete `en_US`/`zh_CN` parity for C17 scope labels and localized
  operational-center card labels.
- Provides one primary `销售开发指挥中心` dashboard provisioner using existing
  extension dashlets and EspoCRM's native `Records` dashlet.

## Integrity

`1.9.8-alpha` is a new deterministic canonical artifact. The prior
`1.9.7-alpha` ZIP and SHA-256 sidecar remain immutable historical evidence.

## Scope disclosure

This alpha release changes presentation, navigation, localization, and
dashboard provisioning only. It adds no workflow, service, ACL, connector,
provider, worker, entity, or schema behavior.
